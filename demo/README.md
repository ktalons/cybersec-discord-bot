## Adding New Assets

1. **Screenshots**: Add `.png` or `.jpg` files
2. **GIFs**: Keep under 2-3 MB for fast loading
3. **Videos**: Consider converting to GIF or hosting externally

## Recommended Tools

- **Recording**: Kap (macOS), OBS Studio, ScreenToGif (Windows)
- **GIF Conversion**: ffmpeg, gifski
- **Optimization**: gifsicle, ImageOptim
